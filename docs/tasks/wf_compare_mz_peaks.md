# W: Compare peaks (interactive, multi-dataset)

## Brief description
Generate HTML document comparing peaks across all datasets.

## Parameters
### **m/zs** (`type` - array)

M/z values.

### **Normalize** (`type` - boolean)

Normalize peaks.
